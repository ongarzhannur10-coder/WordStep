# Vocabulary Learning App

An interactive English vocabulary learning app with AI assistance, gamification, and progress tracking.

## Features

- **Three Proficiency Levels**: A1, A2, B1
- **60 Vocabulary Topics**: 20 topics per level
- **15 Words per Topic**: 900+ words in total
- **Interactive Learning Materials**:
  - Explanations and videos
  - Flashcards for memorization
  - Quiz system with instant feedback
- **Gamification**: Points, levels, and badges
- **Progress Tracking**: Monitor learning statistics
- **AI Assistant**: Ask questions about grammar, vocabulary, and meanings
- **User Authentication**: Secure registration and login

## Tech Stack

- **Frontend**: React.js, Redux, Axios
- **Backend**: Node.js, Express.js
- **Database**: MongoDB, Mongoose
- **Authentication**: JWT, Bcrypt
- **AI**: OpenAI API
